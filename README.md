### 以下参考Java IDE
### C/C++
  \# termux  
  apt install clang  
  \# vim  
  :CocInstall coc-clangd  
### python
  \# termux  
  apt install python  
  pip install jedi  
  \# vim  
  :CocInstall coc-python  
### go-lang
  \# termux  
  apt install golang  
  \# vim  
  :CocInstall coc-go  
### web
  \# termux  
  npm install typescript  
  \# vim  
  :CocInstall coc-tsserver  
### mysql
  \# termux  
  apt install mariadb  
  \# vim  
  :CocInstall coc-sql  
### bash
  \# vim  
  :CocInstall coc-sh  
### vimscript
  \# vim  
  :CocInstall coc-vimlsp
